Awesome choice. The Profile README is perfect because it gives you total creative freedom—you can use images, animations, custom fonts, and styled text layouts.
​Here is exactly how to set it up, along with a creative copy-and-paste template you can use right away.
​Step 1: Create the Secret Repository
​Go to GitHub and click the + icon in the top right, then select New repository.
​Repository name: Type your exact GitHub username. (e.g., if your username is johndoe, the repository must be named johndoe).
​You will see a special green toast message from GitHub saying: "You found a secret! ✨ ...It displays a README.md on your profile."
​Public/Private: It must be set to Public for others to see it.
​Initialize this repository with: Check the box that says Add a README file.
​Scroll to the bottom and click Create repository.
